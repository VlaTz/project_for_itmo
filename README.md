# Time series forecasting
ML-модуль прогнозирования временных рядов

<!--зависимости-->
## Зависимости
Python версии 3.10 или выше. Необходимые библиотеки указаны в файле requiremets.txt

<!--описание коммитов-->
## Описание файлов
| Файл                              | Описание                                       |
|-----------------------------------|------------------------------------------------|
| checking_functions	               | Проверка наличия файлов и ошибок в низ         |
| filling_functions                 | Заполнение результирующих файлов               |
| forecast_demand                   | Последовательное прогнозирование               |
| forecast_demand_multi_input	      | Параллельное прогнозирование                   |
| forecast_functions	               | Основные функции для прогозирования            |
| main_forecast_demand	             | Запуск модуля - для последовательного прогноза |
| main_forecast_demand_multi_input	 | Запуск модуля - для параллельного прогноза     |
| metrics	                          | Метрики оценки качества                        |
| multiprocessing_logging           | Логгирование при работе мультипроцессинга      |
| om_pandas                         | Обработка дат                                  |
| postprocessing	                   | Удаление выбросов в итоговом прогнозе          |
| preparing_functions	              | Подготовка настроек прогнозирования            |
| preprocessing	                    | Подготовка исходных данных к прогнозированию   |
| service_functions	                | Технические функции                            |
| stats_methods	                    | Модели прогнозирвания                          |
